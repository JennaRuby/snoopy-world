# 🐶 Snoopy's World - Web Design Project

A nostalgic and interactive web archive project dedicated to Snoopy and the Peanuts gang, created for the **Web Design course at HYCU**.

## 🚀 Project Overview
This project aims to deliver an engaging user experience by combining nostalgic animation aesthetics with modern web development standards. It features a main archive and dedicated profile pages for iconic characters.

##🛠 Technologies Used
- **HTML5**: Semantic markup for structured and accessible content.
- **CSS3**: Advanced layouts (Flexbox/Grid), Keyframe animations, and custom design variables.
- **Assets**: Optimized GIF archives and localized typography using Google Fonts.

## ✨ Key Features
- **Responsive Design**: Optimized for desktop, tablet, and mobile.
- **SEO & OG Tags**: Professional link previews for social sharing.
- **Web Accessibility**: Descriptive `alt` tags for all visual content.
