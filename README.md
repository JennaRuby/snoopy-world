# Snoopy's World

🚀 **Project Overview**
I developed this Peanuts-themed web archive for my HYCU Web Design class. My goal was to recreate the warm, nostalgic feel of the original Snoopy animation while ensuring the layout is clean and easy to navigate.

## Technologies Used
- **HTML5**: Used semantic tags to create a clear and organized document structure.
- **CSS3**: Applied Flexbox and Grid for the layout, and added custom keyframe animations for the floating clouds.
- **Assets**: Integrated optimized GIFs and Google Fonts to capture the Peanuts aesthetic.

## Key Features
- **Responsive Design**: The layout is optimized to work on mobile, tablet, and desktop screens.
- **SEO & OG Tags**: Included Open Graph tags so that link previews look professional when shared.
- **Web Accessibility**: Added descriptive alt tags to all images to ensure the content is accessible.
